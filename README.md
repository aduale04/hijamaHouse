# website-main
### Required
1) Node js installed
2) MySql Installed with database name defined in .env file
3) Check node version: `node -v`
5) Check npm version: `npm -v`
6) Install typeScript using script `npm i typescript -g`
7) Install yarn using script `npm i yarn -g`

### To run project in Development environment
Run script `yarn start`

### To build for production
Run script `yarn build`

Copy the dist folder created inside root folder
